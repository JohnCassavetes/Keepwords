## Features
- ❌☁️ No iCloud.
- ❌🌱 No Pods
- 👨‍💻100% local.

## Installation
No pods installation. The only thing that makes an app project too complicated is them adding **pods**. Not easy to open.  
```bash
no pods intallation = true
```
